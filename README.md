>   * if 조건문
>   * if else 조건문
>   * 중첩 조건문

```
1. if 조건문
        if (<불 표현식>) {

        }


2. if else 조건문
        if (<불 표현식>) {
            // 불 표현식이 참일 때 실행할 문장
        } else {
        	// 불 표현식이 거짓일 때 실행할 문장
        }


3. 중첩 조건문
        if (불 표현식) {
        	If (불 표현식) {
        		문장;
        	} else {
        		문장;
        	}
        } else {
        if (불 표현식) {
        		문장;
        	} else {
        		문장;
        	}
        }

```
